## Project Overview

This project focuses on predicting crop yield based on weather conditions using machine learning techniques.  
The main goal is to analyze the relationship between weather variables and agricultural production, and then build predictive models that can estimate crop yield more effectively.

The project will be developed and documented through three main phases.

---

## Project Phases

### Phase I – Data Preparation
This phase includes:
- dataset collection
- dataset integration
- data cleaning
- handling missing values
- outlier detection
- data transformation
- preparation of the dataset for machine learning

### Phase II – Model Training and Evaluation
This phase includes:
- selection of machine learning algorithms
- model training
- model evaluation
- interpretation of results
- discussion and justification of the selected techniques

### Phase III – Model Improvement and Comparison
This phase includes:
- improvement of previous models
- feature engineering
- hyperparameter tuning
- comparison of results with earlier phases
- discussion of contributions and conclusions
- recommendations for future work
